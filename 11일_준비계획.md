# 알라딘 업무자동화 포지션 11일 준비계획

## 목표

11일 안에 면접에서 보여줄 수 있는 수준으로 `업무 프로세스 분석`, `CSV/엑셀 자동화`, `간단한 Python 또는 JavaScript 스크립트`, `AI 도구 활용 방식`, `B2B 운영 이해`를 정리합니다.

## 1일차: 채용공고 분석과 자기소개 정리

- 주요업무를 5개 키워드로 정리합니다: 현업 분석, 자동화 발굴, AI 도구, CSV/API/엑셀, 프로토타입 검증.
- 본인의 경험을 위 키워드에 맞게 다시 적습니다.
- 말할 문장: “저는 업무를 먼저 구조화하고, 반복되는 부분을 자동화 단위로 쪼개 빠르게 검증하는 방식으로 일하겠습니다.”

## 2일차: 엑셀/CSV 기본기

- CSV 구조, 컬럼명, 인코딩, 구분자 개념을 공부합니다.
- 엑셀에서 필터, 정렬, 피벗테이블, VLOOKUP/XLOOKUP, IF, COUNTIF, SUMIF를 복습합니다.
- 샘플 주문 데이터를 직접 만들어 상태별 건수를 계산해봅니다.

## 3일차: Python 자동화 기본

- `pandas`로 CSV 읽기, 필터링, 합계 계산, 새 CSV 저장을 연습합니다.
- 핵심 코드 흐름: `read_csv -> 조건 분류 -> 집계 -> to_csv`.
- 면접 답변용으로 “수작업 엑셀 정리를 Python으로 바꾼다면 어떤 순서로 하겠는가?”를 준비합니다.

## 4일차: JavaScript 자동화 기본

- 배열 `map`, `filter`, `reduce`를 공부합니다.
- 현재 포트폴리오의 `app.js`를 읽고 주문 분류 로직을 설명할 수 있게 합니다.
- 버튼 클릭 후 결과가 바뀌는 DOM 조작 흐름을 이해합니다.

## 5일차: API 개념 정리

- API, request, response, JSON, endpoint, status code 개념을 정리합니다.
- 운영툴 API가 있다면 CSV 업로드 대신 API로 주문 상태를 조회할 수 있다고 설명할 수 있게 합니다.
- 예시 답변: “초기에는 CSV로 검증하고, 반복 사용 가치가 확인되면 API 연동으로 자동화 수준을 높이겠습니다.”

## 6일차: 현업 인터뷰 방식 연습

- 현업 담당자에게 물어볼 질문 10개를 작성합니다.
- 예: “이 업무는 하루에 몇 번 하나요?”, “어떤 기준으로 예외를 판단하나요?”, “실수하면 비용이나 고객 영향이 있나요?”
- 자동화 우선순위 기준을 정리합니다: 빈도, 소요 시간, 오류 위험, 규칙 명확성, 연동 난이도.

## 7일차: AI 도구 활용 방식 정리

- ChatGPT/Cursor/Claude Code를 어떻게 쓸지 정리합니다.
- 좋은 사용 예: 요구사항 정리, 테스트 케이스 생성, 코드 초안 작성, 예외 상황 점검.
- 주의할 점: 실제 운영 데이터와 개인정보를 AI 도구에 그대로 넣지 않습니다.

## 8일차: B2B/운영/물류 흐름 이해

- B2B 주문 흐름을 정리합니다: 주문 접수, 재고 확인, 출고 요청, 납품, 정산, 이슈 대응.
- 커머스 운영에서 중요한 지표를 정리합니다: 주문 처리 시간, 출고 지연, 예외 건수, 정산 오류율.
- 알라딘의 도서/상품 유통 맥락에서 납품과 정산이 왜 중요한지 말할 수 있게 합니다.

## 9일차: 포트폴리오 발표 연습

- `index.html`을 열고 3분 발표를 연습합니다.
- 발표 순서: 문제 정의 -> 자동화 대상 선정 -> 데모 -> 확장 방향 -> 본인의 강점.
- `포트폴리오_발표스크립트.md`를 소리 내서 읽고 자연스럽게 줄입니다.

## 10일차: 면접 문답 암기보다 구조화

- `면접_문답집.md`의 답변을 그대로 외우지 말고 핵심 구조만 기억합니다.
- 답변 구조: 상황 -> 판단 기준 -> 실행 방법 -> 기대 효과.
- 모르는 질문이 나오면 “확인 후 작은 프로토타입으로 검증하겠다”는 방식으로 답합니다.

## 11일차: 최종 점검

- 포트폴리오 파일이 정상적으로 열리는지 확인합니다.
- 자기소개 1분, 포트폴리오 3분, 지원동기 1분을 연습합니다.
- 면접 마지막 질문을 준비합니다: “입사 후 가장 먼저 자동화 효과를 기대하는 업무 영역은 어디인지 궁금합니다.”
