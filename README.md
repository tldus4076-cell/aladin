# 알라딘 업무자동화 포트폴리오

알라딘 채용공고의 핵심 요구사항인 `현업 업무 프로세스 분석`, `반복 업무 자동화`, `AI 도구 활용`, `CSV/API/엑셀 기반 자동화`, `빠른 프로토타입 검증`을 보여주기 위한 맞춤 포트폴리오입니다.

## 파일 구성

- `index.html`: 제출/시연용 포트폴리오 웹페이지
- `styles.css`: 포트폴리오 디자인
- `app.js`: 주문/정산 자동화 프로토타입 데모 로직
- `sample-orders.csv`: 자동화 데모용 샘플 주문 데이터
- `11일_준비계획.md`: 11일 안에 준비할 학습/제작 계획
- `면접_문답집.md`: 예상 면접 질문과 답변 방향
- `포트폴리오_발표스크립트.md`: 3분 발표용 스크립트

## 사용 방법

1. `index.html` 파일을 더블클릭해서 브라우저로 엽니다.
2. 화면의 `자동화 결과 생성` 버튼을 눌러 주문/정산 자동화 프로토타입을 시연합니다.
3. 면접 전에는 `11일_준비계획.md`, `면접_문답집.md`, `포트폴리오_발표스크립트.md`를 순서대로 확인합니다.

## 포트폴리오 컨셉

프로젝트명: `B2B 주문/정산 업무 자동화 프로토타입`

목표: 운영 담당자가 반복 수행하는 CSV 주문 확인, 납품 상태 분류, 정산 금액 계산, 예외 건 식별, 현업 공유 메시지 작성을 하나의 간단한 자동화 흐름으로 묶는 것입니다.

## 면접에서 강조할 핵심 메시지

- 개발 자체보다 먼저 현업의 반복 업무 흐름을 구조화했습니다.
- 자동화 가능 업무와 프로덕션 개발이 필요한 업무를 분리했습니다.
- 엑셀/CSV/API/AI 도구를 사용해 빠르게 검증 가능한 프로토타입을 만들 수 있습니다.
- 완성도 높은 대형 시스템보다 현업이 바로 써볼 수 있는 작은 자동화를 빠르게 만드는 데 강점이 있습니다.
